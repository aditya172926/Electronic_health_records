Project Aim

Store and Retrieve the Healthcare records for a patient with a medical institution


Objectives
  • Store and access the healthcare records using Blockchain
  • Decentralize the storage method of the user profiles and data.
  • Remove a single point of failuer
  • Use Inter Planetary File System, a distributed system to store the medical files.
  • Provide a permissioned framework where the documents are sharable and synchronized with multiple entities.

Technologies Used
  • Django Web Framework
  • Python Programming Language
  • HTML, CSS, JavaScript
  • Ethereum Blockchain Protocol
  • IPFS Api wrapper for Python
  • Web3.py Python package
  • Ether.js library for JavaScript
